មេរៀនទី២៖ រូបមន្តត្រីកោណមាត្រ

រយៈពេល៖ ៦០ នាទី ពិន្ទុសរុប៖ ១០០ (៥០ សំណួរ) ១ សំណួរ = ២ ពិន្ទុ
១. តើរូបមន្តព្រលាតនៃ \(\cos(\alpha - \beta)\) គឺជាអ្វី?
ក. \(\cos\alpha\cos\beta - \sin\alpha\sin\beta\)
ខ. \(\cos\alpha\cos\beta + \sin\alpha\sin\beta\)
គ. \(\sin\alpha\cos\beta - \cos\alpha\sin\beta\)
ឃ. \(\sin\alpha\sin\beta - \cos\alpha\cos\beta\)
២. តើរូបមន្ត \(\sin(\alpha + \beta)\) ស្មើនឹង៖
ក. \(\sin\alpha\cos\beta + \sin\beta\cos\alpha\)
ខ. \(\sin\alpha\sin\beta + \cos\alpha\cos\beta\)
គ. \(\sin\alpha\cos\beta - \sin\beta\cos\alpha\)
ឃ. \(\cos\alpha\cos\beta - \sin\alpha\sin\beta\)
៣. គណនាតម្លៃនៃ \(\cos \frac{\pi}{12}\) ៖
ក. \(\frac{\sqrt{6}-\sqrt{2}}{4}\)
ខ. \(\frac{\sqrt{6}+\sqrt{2}}{4}\)
គ. \(\frac{\sqrt{2}-\sqrt{6}}{4}\)
ឃ. \(\frac{\sqrt{3}+1}{2}\)
៤. តើតម្លៃ \(\sin 75^\circ\) ស្មើនឹងតម្លៃត្រីកោណមាត្រមួយណាខាងក្រោម?
ក. \(\sin 15^\circ\)
ខ. \(\cos 15^\circ\)
គ. \(\cos 75^\circ\)
ឃ. \(\tan 75^\circ\)
៥. រូបមន្តមុំឌុប \(\cos 2\alpha\) មិនអាចសរសេរជាទម្រង់ណា?
ក. \(2\cos^2\alpha - 1\)
ខ. \(1 - 2\sin^2\alpha\)
គ. \(\cos^2\alpha - \sin^2\alpha\)
ឃ. \(1 + 2\sin^2\alpha\)
៦. បើ \(\cos\alpha = -\frac{4}{5}\) និង \(\frac{\pi}{2} < \alpha < \pi\) គណនា \(\sin 2\alpha\)៖
ក. \(\frac{24}{25}\)
ខ. \(-\frac{24}{25}\)
គ. \(\frac{7}{25}\)
ឃ. \(-\frac{7}{25}\)
៧. តើរូបមន្ត \(\tan(\alpha + \beta)\) កំណត់បានលុះត្រាតែ៖
ក. \(\alpha, \beta \neq \frac{\pi}{2} + k\pi\)
ខ. \(\tan\alpha\tan\beta \neq 1\)
គ. ទាំង ក និង ខ
ឃ. \(\alpha + \beta = \pi\)
៨. គណនាតម្លៃនៃ \(\tan 105^\circ\) ៖
ក. \(2+\sqrt{3}\)
ខ. \(-(2+\sqrt{3})\)
គ. \(2-\sqrt{3}\)
ឃ. \(\sqrt{3}-2\)
៩. រូបមន្តកន្លះមុំ \(\sin^2 \frac{\alpha}{2}\) ស្មើនឹង៖
ក. \(\frac{1-\cos\alpha}{2}\)
ខ. \(\frac{1+\cos\alpha}{2}\)
គ. \(\frac{1-\sin\alpha}{2}\)
ឃ. \(\frac{1+\sin\alpha}{2}\)
១០. បំប្លែងផលគុណ \(2\sin x \cos y\) ទៅជាផលបូក៖
ក. \(\cos(x+y) + \cos(x-y)\)
ខ. \(\sin(x+y) + \sin(x-y)\)
គ. \(\sin(x+y) - \sin(x-y)\)
ឃ. \(\cos(x-y) - \cos(x+y)\)
១១. តម្លៃនៃ \(\sin 15^\circ \cos 15^\circ\) គឺ៖
ក. \(1/2\)
ខ. \(1/4\)
គ. \(\sqrt{3}/4\)
ឃ. \(1\)
១២. បើ \(\tan \alpha = 2\) គណនា \(\tan 2\alpha\)៖
ក. \(4/3\)
ខ. \(-4/3\)
គ. \(3/4\)
ឃ. \(-3/4\)
... សំណួរទី ១៣ ដល់ ៥០ នឹងត្រូវរៀបចំតាមលំនាំដូចខាងលើ ដោយផ្ដោតលើ៖
  • ការសម្រួលកន្សោមត្រីកោណមាត្រ
  • ការស្រាយបំភ្លឺសមភាពក្នុងត្រីកោណ
  • ការប្រើប្រាស់រូបមន្តបំប្លែងពីផលបូកទៅផលគុណ
  • ការគណនាតម្លៃមុំពិសេស (15°, 75°, 105°, 22.5°)
  • លក្ខណៈនៃមុំបំពេញ និងមុំបន្ថែម
៥០. ក្នុងត្រីកោណ \(ABC\), \(\sin A + \sin B + \sin C\) ស្មើនឹង៖
ក. \(4\cos\frac{A}{2}\cos\frac{B}{2}\cos\frac{C}{2}\)
ខ. \(4\sin\frac{A}{2}\sin\frac{B}{2}\sin\frac{C}{2}\)
គ. \(1 + 4\sin\frac{A}{2}\sin\frac{B}{2}\sin\frac{C}{2}\)
ឃ. \(2\cos A \cos B \cos C\)
